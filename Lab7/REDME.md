# Имитация отжига для задачи коммивояжера (TSP)

## Описание
Реализация метаэвристики "Имитация отжига" для решения задачи коммивояжера на трех языках программирования: Python, C++, Java.

## Задача варианта 7
Реализовать метаэвристику имитация отжига для решения задачи коммивояжера.

## Требования
- Параметры: начальная температура, коэффициент охлаждения, количество итераций
- Использовать 2-opt обмен для построения соседних решений
- Вывести найденный маршрут и его стоимость
- Вывести количество улучшений на различных температурах

## Входные данные
Матрица расстояний для 8 городов
Начальная температура: 100
Коэффициент охлаждения: 0.95

# Алгоритм имитации отжига для TSP

## Теоретическое описание

### Постановка задачи
Задача коммивояжера (TSP) - найти кратчайший маршрут, посещающий все города ровно один раз и возвращающийся в стартовый город.

### Алгоритм имитации отжига
Имитация отжига - метаэвристический алгоритм, основанный на физическом процессе отжига металлов.

### Основные этапы:
1. **Инициализация**: случайный начальный маршрут
2. **Генерация соседних решений**: 2-opt обмен
3. **Критерий принятия**: формула Метрополиса
4. **Охлаждение**: постепенное уменьшение температуры

### Параметры:
- Начальная температура: 100.0
- Коэффициент охлаждения: 0.95
- Итерации на температуре: 100
